# Arcade Fusion

A modern arcade game project built as part of a personal front-end portfolio, focused on energetic visuals, interactive gameplay, and a fast user experience.

## Live Demo

https://laurandreea10.github.io/Arcade-Fusion/

## Features

* Arcade-inspired gameplay
* Interactive front-end experience
* Dynamic pacing and visual energy
* Modern UI styling
* Responsive layout
* GitHub Pages deployment

## Technologies

* HTML
* CSS
* JavaScript
* GitHub Pages

## Purpose

This project was created as a personal web experiment exploring arcade mechanics, visual design, and front-end interactivity in a modern browser-based format.
